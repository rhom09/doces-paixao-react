# Project: Doces Paixão 🧁

## Overview
A professional landing page for a confectionery business, built with React, Vite, TypeScript, and Tailwind CSS. The project uses Sanity.io as a CMS for dynamic content.

## Primary Objective
Complete the current landing page development and launch the site.

## Stack
- **Frontend:** React 19+, Vite, TypeScript, Tailwind CSS
- **State Management:** React Hooks
- **CMS:** Sanity.io
- **Deployment:** Vercel
- **Form Handling:** React Hook Form + EmailJS

## Context
The project is already partially implemented with sections for Hero, Stats, Products, and Contact. Content is being migrated to Sanity for easier management.

## Goals
1. Finalize Sanity integration for all dynamic sections.
2. Ensure full responsiveness and performance optimization.
3. Set up production deployment on Vercel.
